A new django project homepage

Based off of one of the Django Quickstart project templates by Ted Tieken.

As you get started a few things to remember to do

In your Base.html

  • Set Page Title base
  • Set top bar logo-text
  • Set google analytics variables, uncomment google analytics section

In your file directory

  • Add favicons to assets/assets/ico
  • Add your custom css in assets/assets/css/base.css

If this page is displaying without formatting, remember to run 

manage.py collectstatic
Depending on your environment you may need to run it like this: 
      python manage.py collectstatic
      python2.7 manage.py collectstatic
      foreman run python manage.py collectstatic
      heroku run python manage.py collectstatic
